# PUBG Logitech AimFix

Profile based Logitech mouse recoil management for PUBG

# Important

As of April 2026, KRAFTON mentioned in one of their dev letters that they will crack down hard on the recoil script users.
For this i have added 3 noise layers to toggle at will. This does not mask the script 100% and never will, but might make it more confusing for PUBG's anti-cheat.

**Use at own risk! So far i have NOT been banned using this.**

## Features
- Toggle recoil control on/off
- Weapon profile management
- Automatic magazine-based recoil control
- Cycle through weapons with a button press
- Support for burst DMR with recoil control

> **Note:** For information on tuning profiles, see [`profiles`](docs/profiles.md). For examples, take a look at [`examples`](examples/profiles.lua)

## Installation

Copy all the source code in [`recoil`](src/recoil.lua) and paste it into your Logitech script editor.

## Confirmed Mice Support
- G502 Hero
- G502 Lightspeed
